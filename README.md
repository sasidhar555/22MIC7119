# Campus Notifications Frontend

A frontend React application for displaying campus notifications.

## Features

- Search notifications
- Filter notifications
- Responsive UI
- Material UI components
- Icons and hover animations
- Logging middleware
- API integration with fallback data

## Technologies Used

- React
- Vite
- Material UI
- Axios

## Run Project

```bash
npm install
npm run dev
```

## Project Structure

src/
├── components/
├── middleware/
├── pages/
├── services/
